## Creating New Tool Pages

### Quick Start
1. Open `TOOLS_GUIDE.md` for complete instructions
2. Copy the template from `flashcard-generator.html`
3. Replace all placeholders with your tool's content
4. Update meta tags and schema markup
5. Write custom "How to Use" steps
6. Add tool-specific features and description
7. Test on mobile and desktop
8. Add link to `tools/index.html`

### Tools Checklist
Each tool page needs:
- [ ] Unique meta description (80-160 chars)
- [ ] 8-10 relevant keywords
- [ ] Open Graph tags (title, description, image, URL)
- [ ] Schema.org WebApplication markup
- [ ] Proper canonical URL
- [ ] Tool header with gradient background
- [ ] 4-step "How to Use" guide
- [ ] 6 key features with icons
- [ ] Long description (200-300 words)
- [ ] Best practices list
- [ ] 3 student testimonials
- [ ] 3 sidebar widgets
- [ ] Call-to-action button
- [ ] Mobile responsive design
- [ ] Tested on multiple devices

## Troubleshooting

### Common Issues

**Pages not appearing in search:**
- Check meta robots tags
- Verify canonical URLs
- Submit sitemap to GSC
- Wait 2-4 weeks for indexing

**Mobile layout broken:**
- Test with DevTools
- Check viewport meta tag
- Verify media queries
- Test on real devices

**Schema markup not validating:**
- Use Google Rich Results Test
- Check JSON formatting
- Verify all required fields
- Update syntax if needed
