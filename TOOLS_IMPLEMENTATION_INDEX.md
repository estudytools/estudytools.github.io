## Quick Reference Guide

### Creating a New Tool Page

**Step 1: Template**
- Copy `/tools/flashcard-generator.html`
- Use as complete template

**Step 2: Customize**
- Replace tool name (5 places)
- Update meta tags
- Change color gradient
- Write custom content

**Step 3: Sections to Write**
1. Meta description (80-160 chars)
2. Keywords (8-10)
3. How to Use (4 steps)
4. Features (6 items)
5. Long description (200-300 words)
6. Testimonials (3 quotes)
7. Sidebar stats (3 widgets)

**Step 4: Test**
- Mobile (375px, 768px)
- Desktop (1920px)
- Links work
- Images load
- Schema validates
- Lighthouse 90+

**Step 5: Deploy**
- Save to `/tools/[tool-name].html`
- Add link to `/tools/index.html`
- Update sitemap.xml
- Submit to Google

**Time**: 20-30 minutes per tool
