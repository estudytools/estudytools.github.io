## Installation

1. Clone the repository
```bash
git clone https://github.com/estudytools/estudytools.github.io.git
```

2. Navigate to the directory
```bash
cd estudytools.github.io
```

3. Open `index.html` in your browser or deploy to GitHub Pages

## Development

### Making Changes
1. Edit HTML files directly
2. Modify `styles.css` for styling
3. Update `script.js` for functionality
4. Test on multiple devices

### Adding New Tools
1. Add tool data to `tools` array in `script.js`
2. Update tool descriptions
3. Update count in page headers
4. Run through all pages to verify

## Deployment

The site is deployed on GitHub Pages at `estudytools.github.io`.

To deploy your own version:
1. Fork the repository
2. Rename to `[username].github.io`
3. Push changes
4. Site automatically deploys at `https://[username].github.io`
